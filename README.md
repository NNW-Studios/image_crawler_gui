# Image Crawler Downloader

Kleine Tkinter-App zum Crawlen einer Website und zum Herunterladen gefundener Bilddateien.

## Dateien

- `image_crawler_gui.py` startet die grafische Anwendung.
- `run_image_crawler.bat` startet das Script unter Windows.

## Hinweise

- Die lokale Konfiguration wird im Benutzerprofil gespeichert und nicht im Repo.
- Downloads sind auf 25 MB pro Datei begrenzt, damit keine extrem grossen Dateien versehentlich geladen werden.
- Bitte nur auf Websites verwenden, fuer die du die Erlaubnis hast.

## Start

```bat
run_image_crawler.bat
```
